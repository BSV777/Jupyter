fuzzyTECH Bibliothek: fuzzyTECH Installation

Von dieser Seite aus können Sie fuzzyTECH Demo- und Vollversionen mit der dazugehörenden Dokumentation direkt installieren. Folgende Software ist als Download verfügbar:
- fuzzyTECH Release 5.12c vom 23. September 1998 (32 bit)
- fuzzyTECH Demo Release 4.22 (16 bit)

fuzzyTECH Release 5.12 (32 bit)

Installieren Sie das fuzzyTECH Release 5.12c vom 23. September 1998 jetzt (FT512C.EXE: 12.1 MB, Installationsdatei)
32-bit Version for MS-Windows95+ and MS-Windows/NT4.0+. Die Installationsdatei enthält alle fuzzyTECH Editions sowie die Zusatzmodule NeuroFuzzy und FuzzyCluster. Ab Release 5.1x von fuzzyTECH wird es keine eigenständige "Demo" Version mehr geben. Jede fuzzyTECH Edition schaltet automatisch in den Demo-Modus um, falls Sie keine Lizenzrechte zur Benutzung einer Vollversion haben. Beachten Sie, daß Sie nun mehrere fuzzyTECH Editionen auf einmal installieren können.

Bei der Installation gehen Sie folgenderweise vor: 1.) Speichern Sie die Datei in ein Verzeichnis auf Ihrem lokalen PC. 2.) Rufen Sie dann die selbstentpackende Datei FT512C.EXE auf. 3.) Rufen Sie im Verzeichnis Disk1 die Installationsdatei SETUP.EXE auf und folgen Sie dann den während der Installationsroutine gegebenen Anweisungen.

Falls Sie eine ältere Version von fuzzyTECH aktualisieren möchten, so geben Sie den Installationscode ein, der auf Ihrer fuzzyTECH Installations-CD-ROM aufgedruckt ist. Falls Sie fuzzyTECH nur mit Demo-Funktionalität installieren möchten, so geben Sie "000000" um die fuzzyTECH Produkte für technische Anwendungen zu installieren oder "123456" um fuzzyTECH Produkte für betriebs- und finanzwirtschaftliche Anwendungen zu installieren.

fuzzyTECH Demo Release 4.22 (16 bit)

fuzzyTECH Demo 4.22 (16-bit Version) jetzt herunterladen (6,7 MB, selbstextrahierende Archivdatei)
16-bit Version für MS-Windows 3.1x, WIN-OS/2, MS-Windows 95, MS-Windows/NT 3.51 & 4.0. Diese Software enthält 16-bit und 32-bit Komponenten und kann mit jeder anderen 16/32-bit Software auf 16-bit oder 32-Betriebssystemen zusammen benutzt werden. Bei der Installation können Sie zwischen der "fuzzyTECH Edition Demo" (technische Anwendungen) und "fuzzyTECH for Business Demo" (betriebs- und finanzwirtschaftliche Anwendungen) durch Angabe eines Codes während der Installation wählen. Die "Edition Demo" mit dem Installationscode "000000" enthält 10 Beispielprojekte regelungstechnischer Anwendungen. Die "for Business Demo" mit dem Installationscode "123456" enthält 8 Beispielprojekte betriebswirtschaftlicher Anwendungen.
(Installation: Falls Sie vom Browser gefragt werden, was Sie mit der Datei FTDEMO42.EXE machen wollen, so speichern Sie diese in ein temporäres Verzeichnis auf Ihrem PC, rufen dann diese dort auf (Doppelklick im Explorer) und lesen die Datei README.TXT, die beim Extrahieren erzeugt wurde.)
