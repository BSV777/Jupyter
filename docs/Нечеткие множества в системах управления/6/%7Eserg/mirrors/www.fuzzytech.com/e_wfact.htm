fuzzyTECH: Simulation Module

The Simulation Module is a stand-alone simulation module that uses fuzzyTECH as server for fuzzy computations. If you have any other simulation software product, you can connect it to fuzzyTECH by using one of its integrated interfaces. However, if you don't have another simulation software tool yet, we would like to invite you to take a closer look at fuzzyTECH's Simulation Module that consists of the WinFACT® software package by Dr. Kahlert Corporation and a fuzzyTECH-based fuzzy logic function block. This function block allows for the computation of any fuzzy logic system within WinFACT, with or without the fuzzyTECH system. If used with the complete fuzzyTECH system, the entire fuzzy inference flow shown in fuzzyTECH's editors and analyzers reflects the data analysis process in real time. If used without the complete fuzzyTECH system, WinFACT can compute a fuzzyTECH generated stand-alone system.

WinFACT comprises a number of individual program modules that can be combined. The components of WinFACT enable system analysis, control system synthesis, and block oriented simulation of control loops. Interface drivers for periphery PC boards (A/D-D/A converter, etc.) are available as options that let you link WinFACT directly to real-world processes. For implementation on target hardware platforms, a C code generator is available as an option.


WinFACT's Main Module is the Block Oriented Simulation Module BORIS

Block Oriented Simulation

To configure a block oriented simulation diagram, the standard system blocks are positioned at arbitrary positions on the screen and linked by point-and-click. The build-in autorouter places the signal lines automatically. The diagrams can be scrolled, and meta blocks can be created out of standard system blocks to structure even complex systems transparently. An overview function lets you always see the entire picture. The only limit for your system's complexity is the memory and computing power available on your PC. While WinFACT comes with all the standard system blocks required for control engineering simulation and virtual instruments for visualization, you may create your own system blocks as DLLs and add them to the WinFACT system block library.
