FUZZY LOGIC SOFTWARE NOW AVAILABLE FOR
ALLEN-BRADLEY CONTROL COPROCESSOR

Contact: Mike Moore, Allen-Bradley Company, Inc., (216) 646-3094

DETROIT, April 11, 1994 - One of the world's leading fuzzy logic software tools from Inform Software Corporation is now available for Allen-Bradley Control Coprocessor applications.

Inform's fuzzyTECH® is an object-oriented user interface that supports all design steps for fuzzy system engineering. fuzzyTECH provides a front-end graphical interface for program development and process optimization, while at the same time automatically generating the intelligent code for the Control Coprocessor. In addition, fuzzyTECH also features real-time visualization capabilities to optimize a running system "on-the-fly." Adaptive system design is supported by fuzzyTECH's NeuroFuzzy software that integrates neural network technologies.

Industrial automation leader Allen-Bradley Company, Inc., a Rockwell Automation business, announced that fuzzyTECH has been chosen as a selected product in its Pyramid Solutions Program. This program is dedicated to finding leading hardware and software products which help add customer value to Allen-Bradley automation control systems.

"With fuzzyTECH now available for the Allen-Bradley Control Coprocessor, users have another major software tool to help provide control process application solutions in new and stronger ways," said Amy Archer, Control Coprocessor Product Manager. Currently, the Control Coprocessor runs programs in C, BASIC, and assembler languages in real-time multitasking operating system environments.

Archer said that users will want to use fuzzyTECH in a number of analog-based process applications. fuzzyTECH fine tunes a process based on the user's application expertise versus mathematical or control algorithm modeling. A system is configured by descriptive rules rather than by algebraic equations.

She said that this object-oriented interface is extremely easy to learn and exceptionally good at solving process oriented analog-based applications. The standard software tool, fuzzyTECH, has been created by Inform, a company with 24 years experience in fuzzy logic research and application. Inform is a market leading company in these technologies. Inform's technical teams have served as consultants on many successful fuzzy logic applications worldwide.

Constantin von Altrock, Director of Inform's Fuzzy Technologies Division, said, "Due to the flexibility and computing power of the PLC-5® programmable controller and its Control Coprocessor, even large fuzzy logic systems can be computed in milliseconds."

Allen-Bradley, a Rockwell Automation business, is a worldwide leader in the manufacture and sale of industrial automation controls and systems. Allen-Bradley has its world headquarters in Milwaukee, Wis. Rockwell is a diversified, high technology company holding leadership positions in automation, avionics, aerospace, defense electronics, telecommunications, automotive components and graphic systems, with annual worldwide sales of $11 billion.
