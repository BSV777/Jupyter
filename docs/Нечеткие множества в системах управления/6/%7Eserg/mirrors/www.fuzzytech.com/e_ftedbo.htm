Rapid Application Development

In addition to its standard functionality for creating fuzzy logic systems, fuzzyTECH features a set of Fuzzy Design Wizards that:
- enable rapid application design,
- let you conduct ultra-fast prototyping, and
- guide you through the IEC/ISO standardized fuzzy logic design approach.

There are three Fuzzy Design Wizards that can lead you though all design phases to a fuzzy logic solution: 1. Inference Structure Wizard, 2. Linguistic Variable Wizard, and 3. Fuzzy Rule Wizard. Each Fuzzy Design Wizard poses a series of questions to the designer that lead up to a complete prototypical fuzzy logic system:
- Novice users of fuzzyTECH can generate prototype fuzzy logic systems within minutes.
- Experienced designers use it to expedite and streamline their design process.

Inference Structure Wizard Linguistic Variable Wizard  Fuzzy Rules Wizard
The Inference Structure Wizard, also dubbed "Fuzzy Design Wizard", creates a complete fuzzy logic system structure containing linguistic variables, membership functions, and fuzzy rules. It can even evaluate sample data files which represent the system to be designed. It uses the result of this analysis to propose answers to the design questions. While the Inference Structure Wizard creates the terms of a linguistic variable definition, the Linguistic Variable Wizard lets you create complete sets of consistent membership functions in one simple step. The Fuzzy Rule Wizard implements a structured audit to create a rule base from engineering experience. This rule base forms the starting point for further optimization.

ISO 9000 Compliant Development

The structured design approach of fuzzyTECH is complemented by a built-in documentation generator and a revision control system. The documentation generator generates complete documentation for ISO 9000 compliant development. You can view, print, and edit the RTF output with standard word processors. If your company uses standard formats for documentation, you can define your personalized documentation format using most standard word processing software. You can also establish how detailed the documentation generator should be when explaining your design. Settings can range from a brief table-and-graphs type format to a complete explanation of the individual fuzzy logic components. The documentation generator even covers development history using the integrated revision control system of fuzzyTECH. The revision control system lets you rewind to earlier development stages and can document the progress. The entire structure of documentation follows the IEC 1131-7 standard on fuzzy logic development.

Runtime Target Support

fuzzyTECH can utilize its internal computation engine to compute the fuzzy logic inference, letting you view the inference process graphically in its editors and analyzers. While you may choose to use fuzzyTECH for computing your final fuzzy logic solution (for example, fuzzyTECH tied to an Excel spreadsheet), you can also deploy a fuzzy logic system as a stand-alone runtime system. The benefits of this option are:
- you can freely distribute the runtime system (no runtime license required)
- the computing performance of the runtime system is superior
- your rules and membership functions are hidden in the system
- fuzzyTECH offers runtime solutions for many hardware platforms

The technologies used to implement runtime solutions on target hardware differ greatly:
- for PC implementations, a DLL/OLE runtime module is available
- for microcontroller implementations, assembly code generation is available
- for programmable logic controller (PLC) implementations, function blocks are available
- for process control systems, distributed fuzzy logic modules are available
- for generic target systems, ANSI-C source code generation is available
To best suit the needs of the different target hardware platforms, a number of different editions of fuzzyTECH are available. The next page presents an overview of the different fuzzyTECH Editions.

Online/RTRCD Debugging

The "Online" debugging technique allows for the modification of a running fuzzy logic system "on-the-fly". All editors and analyzers of fuzzyTECH let you monitor the fuzzy logic inference in real time and every modification is implemented instantly on the target hardware.

fuzzyTECH can communicate bi-directionally with the target system at any time via a serial cable, a network file system, or another communication link. The real-time data is transferred from the target hardware to the graphical editors and analyzers of fuzzyTECH, allowing complete visualization of the fuzzy logic inference. Any modification to the fuzzy logic system is transferred from fuzzyTECH to the target hardware and implemented without halting this system. Remote traces can be configured and controlled from fuzzyTECH as well.

Some fuzzyTECH MCU Editions also provide a limited online debugging functionality, called RTRCD (Real-Time Remote Cross Debugging). RTRCD lets you modify rules and membership functions on-the-fly and run traces.
