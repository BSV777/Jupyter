fuzzyTECH: DataAnalyzer Module

The fuzzyTECH DataAnalyzer Module supplies data processing and analysis functionalities to support fuzzy logic solutions in sensor signal processing, data analysis, and classification. This includes any kind of preprocessing of sensor data before the fuzzy logic computation, and multiple analyzer features for visualizing the results of the fuzzy logic computation. The DataAnalyzer Module can be used together with the NeuroFuzzy Module and all other add-on modules.

The DataAnalyzer Module consists of the DIAdem® software package by GfS Corporation and a fuzzyTECH-based fuzzy logic function block. This function block allows for the computation of any fuzzy logic system within DIAdem, with or without the fuzzyTECH system. If used with the complete fuzzyTECH system, the entire fuzzy inference flow shown in fuzzyTECH's editors and analyzers reflects the data analysis process in real time. If used without the complete fuzzyTECH system, DIAdem can compute a fuzzyTECH generated stand-alone system.


The DataAnalyzer Module Lets You Develop and Run Complete Fuzzy Data Analysis Applications Without Programming a Single Line of Code

DIAdem lets you develop and implement an entire data analysis system by connecting and configuring function blocks. The function block structure is developed graphically, with fuzzyTECH providing fuzzy logic function blocks.

For data processing, multiple blocks for spectrum analysis and digital filtering, as well as mathematical and statistical functions are provided. For visualization, oscilloscopes, meters, and plots can be defined in multiple configurations. Special blocks interface to peripherals, such as I/O boards, serial ports, and other software modules. The function block library can be expanded by user-programmed DLLs.
