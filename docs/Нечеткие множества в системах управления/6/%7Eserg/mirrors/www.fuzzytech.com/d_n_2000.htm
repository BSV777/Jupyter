INFORM GmbH gibt die "Year 2000 Compliance" für alle fuzzyTECH Produkte bekannt

Mit fuzzyTECH ins Jahr 2000! Alle fuzzyTECH Produkte, die ab Juli 1998 (ab Release 5.11a) ausgeliefert wurden, sind unter MS Windows 95/98 oder unter MS Windows NT 4.x kompatibel für das Jahr 2000.

Schriftliche Garantie

Alle Anwender von fuzzyTECH, die einen Updatevertrag abgeschlossen haben, der über den 1. Januar 2000 hinausgeht, erhalten auf Wunsch eine schriftliche Bestätigung und Garantie der Year 2000 Compliance.

Wie Sie eine "Year 2000 Compliant" Version von fuzzyTECH erhalten können

Wurde Ihr fuzzyTECH 5.1 zwischen dem 1. Januar 1998 und dem 1. Juli 1998 ausgeliefert, so können Sie ein Gratisupdate auf das aktuelle Wartungsrelease von diesem Web-Server installieren. Wurde Ihr fuzzyTECH vor dem 1. Januar 1998 ausgeliefert, oder haben Sie ein fuzzyTECH vor dem Release 5.1, so benötigen Sie ein Direktupdate. Bitte fordern Sie ein Angebot direkt von uns an.
