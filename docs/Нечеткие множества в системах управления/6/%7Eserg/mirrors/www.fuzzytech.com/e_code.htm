fuzzyTECH Generated Code Examples  

fuzzyTECH offers multiple ways to implement a designed fuzzy logic system:
- generation of portable C source code
- generation of microcontroller specific assembly code
- generation of binary files for the stand-alone runtime modules

This page lets you view examples of generated C source code, generated Assembly source code, and the FTL file format used by fuzzyTECH to store projects on disk. All three examples use the simple BECH0 benchmark project. For more information on BENCH0 refer to the Bechmark section. Notice that only the project specific code generated by fuzzyTECH is shown. This code contains the code representation of the fuzzy logic system and calls computation routines that are provided in source with the respective fuzzyTECH Edition. Also notice that simple code generation options have been used, for example, all optimizers and the Online/RTRCD code generation options have been disabled.
