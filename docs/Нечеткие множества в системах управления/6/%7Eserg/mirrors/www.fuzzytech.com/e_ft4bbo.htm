fuzzyTECH for Business / Basic Overview

fuzzyTECH for Business Features:

  • Complete Development Software for Fuzzy Logic in Finance and Business Applications
  • Full Graphical Development and Visual Optimization
  • Integrated NeuroFuzzy Module for Automated System Generation
  • Easily Links to Standard Business Software
  • Advanced Fuzzy Inference and Compensatory Operators

Fuzzy Logic provides a solution for business and financial applications based on human experience and judgment. In addition, fuzzy logic lets you deal with uncertainty and imprecise data. A dedicated development tool is a prerequisite to efficiently design such applications.

With fuzzyTECH for Business, INFORM provides such a development platform. fuzzyTECH for Business is based on the fuzzyTECH Editions line of development tools that over the past five years has become a standard in technical applications of fuzzy logic. fuzzyTECH for Business is the only fuzzy logic tool that provides seamless integration with standard software packages such as MS-Excel, MS-Access or VisualBasic. fuzzyTECH for Business can also be used as a stand-alone application or in a mainframe environment. fuzzyTECH for Business supports all open standards for integration and portability. It uses the FTL format to store fuzzy logic projects on disk. The FTL format is a fuzzy logic system description format that is supported by many hardware and software vendors. Also, fuzzyTECH features standard MS-Windows interface technologies such as DLL or DDE.

Automated System Generation

fuzzyTECH for Business comes with a built-in NeuroFuzzy Module for automated system Generation. This module utilizes neural network techniques to create and optimize membership functions and fuzzy logic rules from existing sample data. The result of the NeuroFuzzy training is a fuzzy logic system that can be verified and implemented using all fuzzyTECH analyzers and tools.

Enhanced Fuzzy Technologies

fuzzyTECH for Business supports all enhanced fuzzy technologies that have proven useful in real-world applications. In particular, fuzzyTECH supports:

  • seven (7) different defuzzification types,
  • all three (3) families of compensatory operators in parametric form, and
  • all eight (8) types of membership functions in parametric form.

In addition, the fuzzy inference can be adapted to different application needs.

Fuzzy Design Wizards

Using the rich feature set of fuzzyTECH is a breeze with the new Fuzzy Design Wizards. For new users of fuzzy logic, the Fuzzy Design Wizards provides a step-by-step guide to fuzzy logic design methodology. Experienced users can create prototypes of even highly complex systems in just minutes.

Furthermore, the Fuzzy Design Wizard also analyzes sample data files and uses acquired information in proposing design decisions. The optional Fuzzy Rule Wizard automatically creates complete rule bases from an interactive audit session, and the optional Linguistic Variable Wizard lets you create even complex membership functions with a few mouse clicks.

Worldwide Support

fuzzyTECH for Business is based on INFORM's twenty years of experience in building real-world business and financial applications. INFORM's risk assessment system "ASK" was the first commercial fuzzy logic decision support system in finance in the world.

Today, INFORM's project teams implement fuzzy logic applications in logistics, production control, supplier evaluation, stock and exchange market evaluation, and other applications.
