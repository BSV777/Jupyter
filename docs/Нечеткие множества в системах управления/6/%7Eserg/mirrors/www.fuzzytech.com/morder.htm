User Interface
All fuzzyTECH products share the same user interface with all-visual editor and analyzer capabilities. For an overview, refer to the general fuzzyTECH Overview.

Code Generation
All fuzzyTECH Explorer and MCU Editions let you generate both portable C code and processor specific assembly code (code examples).

Toolkit Integration
In close cooperation with Compiler/Assembler vendors, Inform ensures smooth integration of fuzzyTECH with Cosmic, Hiware, and Motorola MCU toolkits. You can switch toolkit integration any time during design (screen shot of toolkit integration menu).

Compatibility
A fuzzy logic design created with one fuzzyTECH product can be opened and further developed with any other fuzzyTECH product because fuzzyTECH uses the open FTL (fuzzy technology language) file format to store fuzzy logic projects.

Software Update Contract, Direct & Cross Updates

With the Software Update Contract, a new release of your fuzzyTECH products (Editions, Modules, Packages) is automatically sent to you as soon as it becomes available. The Software Update Contract is valid for one year minimum and is automatically extended, if not canceled. Availability limited at some locations. A registered user of fuzzyTECH may purchase an Direct Update to the current release of the same fuzzyTECH product at any time. Please ask for a quotation. To switch to a different Edition of fuzzyTECH, Cross Updates are available. Please ask hotline@inform-ac.com for a quotation.

Release Information

The current release of all fuzzyTECH products is 5.10 (32-bit). This release has shipped since January 20, 1998. It has been thoroughly tested with Windows95 and Windows/NT 4.0. This release is a pure 32-bit application and does not run with 16-bit operating systems, such as MS-Windows 3.11. For MS-Windows 3.1x, Windows/NT 3.51, and Win-OS/2 3.0 we still provide the release 4.2 (16-bit) of fuzzyTECH. Both releases provide interfaces for both 16-bit and 32-bit software. For a list of new features, visit the Release Information section.
