fuzzyTECH Library: fuzzyTECH Installation

From this page, you may install fuzzyTECH demos and full versions as well as documentation and related information. The following software is available for download:
- fuzzyTECH Release 5.12c from September 231, 1998 (32-bit)
- fuzzyTECH Demo Release 4.2 (16-bit)

fuzzyTECH Release 5.12 (32-bit)

Install fuzzyTECH Release 5.12c from September 23, 1998 now (FT512C.EXE: 12.1 MB, Installation file)
32-bit Version for MS-Windows95+ and MS-Windows/NT4.0+. The installation file contains all fuzzyTECH Editions as well as the NeuroFuzzy and FuzzyCluster add-on Modules. With Release 5.1+ of fuzzyTECH, there is no dedicated "Demo" version anymore. Rather, every fuzzyTECH Edition switches to Demo functionality if you do not have the license right to run the full version. Notice that you can install multiple fuzzyTECH Editions in one step.

Installation steps: 1.) Save the file to your local drive. 2.) Execute the self-extracting file FT512C.EXE. 3.) Open the Disk1 directory and execute there the installation file SETUP.EXE. 4.) Follow then all further instructions given during the setup routine.

If you are updating from an older version of fuzzyTECH, use the Installation Key printed on your CD-ROM or setup disk. If you would like to install fuzzyTECH in demo functionality, enter "000000" to install the fuzzyTECH products for technical applications and "123456" to install fuzzyTECH products for business and finance applications.

fuzzyTECH Demo Release 4.22 (16-bit)

Download fuzzyTECH Demo Release 4.22 archive now (FTDEMO42.EXE: 6.7MB, self-extracting archive file)
16-bit Version for MS-Windows 3.1x, WIN-OS/2, MS-Windows 95, MS-Windows/NT 3.51 & 4.0. This release contains both 16-bit and 32-bit components and can interface with any other 16- or 32-bit software program on both 16-bit and 32-bit operating systems. You can choose between the "fuzzyTECH Edition Demo" (technical applications) and the "fuzzyTECH for Business Demo" (business and finance applications) by entering the respective installation code during installation. The installation code for the "Edition Demo" is "000000" and the installation code for the "for Business Demo" is "123456". A demo version of the NeuroFuzzy Module is included and will be installed with the Demo. The "Edition Demo" contains 10 sample projects of control engineering applications, and the "for Business Demo" contains 8 sample projects of business and finance applications.
(Installation: When your browser asks you what to do with the file FTDEMO42.EXE, save it in a temporary directory, then execute it and read README.TXT)
