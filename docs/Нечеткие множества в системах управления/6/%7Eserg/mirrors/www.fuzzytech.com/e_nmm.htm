fuzzyTECH: NeuroModel Module

fuzzyTECH's NeuroModel Module is a dedicated software add-on to fuzzyTECH for the process control industry. It uses neural network technology to build a plant model based on recorded process data. fuzzyTECH's NeuroModel Add-On Module consists of the complete NeuroModel® software package by Atlan-tec Corporation and a direct interface to fuzzyTECH.


NeuroModel Uses Process Data to Generate a Model for the Process Behavior
that can be Used to Design a Fuzzy Logic Controller with fuzzyTECH

NeuroModel bases on neural net technology and statistical optimization methods to convert recorded process data into a continuous model of the process behavior. This model can be used in multiple ways. In a direct combination with fuzzyTECH, the neural model replaces a process simulation. In complex processes where the mathematical model used for the simulation is too hard to derive, a neural process model provides an efficient alternative to conventional simulation techniques.

NeuroModel uses an expert system to configure and parameterize the neural net itself and its training parameters. Unlike other neural net software tools, NeuroModel does not require complex parameterization by the user. To verify the generated neural process model and to proof its consistency with the process, NeuroModel features a number on build-in analyzer tools. The patended SecurityNet® technology identifies areas that require re-training or intervention. 2D/3D/4D plots visualize the interdependencies between the model parameter for further in-depth analysis.
