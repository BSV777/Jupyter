TI AND INFORM ANNOUNCE FIRST FUZZY LOGIC TOOLS
FOR DSP-BASED SYSTEM DEVELOPMENT

Contact: Donna Coletti, (713) 274-3361

HOUSTON (Jan. 19, 1994)-- Designers of digital signal processing (DSP)-based systems will now be able to implement fuzzy logic techniques because of new software available from Texas Instruments and Inform Software Corporation. Fuzzy logic, when used in DSP-based applications, merges human understanding with traditional control techniques by using elements of everyday language to represent desired system behavior. This circumvents the need for rigorous mathematical modeling. More intelligent control, in addition to increased performance, can be incorporated into products ranging from hard disk drives to motors to radar and sonar systems. These products are easier to use because certain functions can now be automated for the first time.

TMS320 fuzzyTECH Tools

The new TMS320 fuzzyTECH logic tools include fuzzyTECH MCU-320 Edition and fuzzyTECH MCU-320 Explorer. The fuzzyTECH MCU-320 Edition is the first complete software development environment for fuzzy logic systems implementing DSPs. The fuzzyTECHMCU-320 Explorer is a fuzzy logic system design training tool based on MS-Windows™. Both are designed for use with TI TMS320C2x and TMS320C5x generation DSPs.

The fuzzyTECH MCU-320 Edition generates portable C code and optimized assembly code providing designers with easy integration to conventional code. Graphical tools are provided for all development steps including design, optimization and verification. At the push of a button, the built-in code generator outputs the entire fuzzy system optimized for the selected TI `C2x or `C5x DSP.

The fuzzyTECH MCU-320 Edition is compatible with a traditional fuzzy design flow and all other non-TMS320 versions of fuzzyTECH. System designs are represented in fuzzy technology language, making designs hardware independent. This independence provides users flexibility in determining system configurations and the processors necessary to meet system performance requirements. The fuzzyTECH MCU-320 Explorer is a computer-assisted training software program that teaches the design steps of fuzzy logic-based systems.

Fuzzy Logic as a Control Technique

Fuzzy logic offers an alternative solution to conventional control techniques. Because DSPs are architecturally optimized for mathematically-intensive tasks, a single DSP can accomplish both signal processing functions, such as filtering and signal estimation, as well as performing control tasks using either conventional control techniques using proportional integrated differential and/or fuzzy logic. The fuzzyTECH tools will have the most significance in the high-performance arena, including compression applications involving video and voice, applications requiring very fast control, such as in hard disk drives and motors, and signal analysis applications in equipment, such as radar and sonar.

"Fuzzy logic has been proven to be a successful technology in both Europe and Japan in a board range of applications ranging from appliances and automotive systems to industrial automatization and signal processing," said Constantin von Altrock, director of the Fuzzy Technologies Division at Inform. "By using fuzzy logic, these products have been designed in less time, yielding more functionality and higher performance."

"Our benchmark shows that TMS320 DSPs, when implementing fuzzy logic, provide ten times the performance over a comparably priced conventional 16-bit microcontroller," said Dr. Kun-Shan Lin, TI´s DSP marketing manager. "Similar performance levels have also been obtained in comparisons with hardwired fuzzy chips, which do not have the inherent flexibility of a programmable DSP. Now, non-DSP users have an option which delivers flexible, improved performance for next generation designs."

Availability

The TMS320 fuzzyTECH logic tools are currently available in beta versions from TI. Production availability is planned for early December from TI, Inform and authorized distributors. Suggested retail price for the fuzzyTECH MCU-320 Edition is $1,890. Suggested retail price for the fuzzyTECH MCU-320 Explorer is $199.

The TMS320 DSP family has led the industry since TI introduced ist first member in 1982. Today, the TMS320 family is the most widely used family of single-chip DSPs and includes five generations of more than 30 compatible devices, providing designers with both fixed- and floating-point devices to fit a variety of price/performance requirements. Applications range from consumer products to telecommunications switching equipment to high-reliability military applications to active suspension in automobiles.

Inform Software Corporation of Illinois and INFORM GmbH of Germany have been involved in extensive research and development of fuzzy technologies for over a decade years. Inform has developed a variety of fuzzy industrial applications worldwide over the years and brings a wealth of real-world expertise to the design of its fuzzyTECH tools and embedded designs.

TI`s Total Integration™ capabilities attract teacher-costumers from around the world giving TI system expertise to enable differentiation of its costumers' products in key end equipment market segments, including computer, communications, consumer electronics, automotive, military and industrial.
