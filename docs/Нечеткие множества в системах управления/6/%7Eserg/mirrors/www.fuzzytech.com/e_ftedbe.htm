fuzzyTECH: 4. Benchmarks

The computational speed and memory requirements for five (5) different standardized fuzzy logic systems are given in Table 6. These fuzzy logic systems (Bench 0 to Bench 4) stem from industrial fuzzy logic applications and represent typical configurations:

  • Bench 0: Simple Fuzzy Logic Position Controller
    (7 Rules, 2 Inputs with 3 terms each, 1 Output with 5 terms, all rules weights = 1)
  • Bench 1: Fuzzy PI-Controller
    (20 Rules, 2 Inputs with 5 terms each, 1 Output with 5 terms, all rules weights = 1)
  • Bench 2: Fuzzy PD-Controller
    (20 Rules, 2 Inputs with 5 terms each, 1 Output with 5 terms, various rules weights)
  • Bench 3: Parameter Estimator in an ABS System
    (80 Rules, 3 Inputs with 3, 4, and 6 terms, 1 Output with 5 terms, various rules weights)
  • Bench 4: Intervehicle Dynamic Controller
    (500 Rules, 8 Inputs with 7 terms each, 4 Outputs with 7 terms each, various rules weights)

Bench 4 stems from a research application and is the largest known embedded fuzzy logic system. All computing times are given as average / maximum (average case / worst case). These benchmarks are included as FTL source with every fuzzyTECH Edition. You can also download these Benchmark systems from this WWW site as FTL source code.
