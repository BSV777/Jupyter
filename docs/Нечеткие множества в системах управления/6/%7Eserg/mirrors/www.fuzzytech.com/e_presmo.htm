MOTOROLA AND INFORM RELEASE FUZZY LOGIC TOOLS FOR 68HC11 ABD 68HC12 FAMILIES

Editorial Contact: Greg Evans, Cunningham Communication, Inc., (512) 347-2305

AUSTIN, Texas -- October 6, 1997 -- Today, Motorola's Semiconductor Products Sector and Inform Software Corp. announced the availability of new fuzzy logic development tools for the 8-bit 68HC11 and 16-bit 68HC12 microcontroller families that can simplify programming, reduce code size and allow code to execute faster. The 68HC12 is the first general purpose microcontroller family with dedicated fuzzy logic instructions.

Developed with Inform Software Corp., the leading fuzzy logic technology developer, these new tools ease the development of customer's Motorola microcontroller-based products which utilize fuzzy logic. The two companies will also combine efforts to educate the design community regarding the benefits of fuzzy logic technology.

"Inform is the leading supplier for fuzzy logic development tools," said Greg White, general manager of Motorola's Safety & Chassis Systems Division. "We are excited to work with them to offer the best fuzzy logic development tools for the 68HC11 and 68HC12 microcontroller families."

"Motorola's 68HC12 is the first standard microcontroller family with a comprehensive fuzzy logic instruction set and, as such, is the premier 16-bit engine for executing fuzzy logic," said Constantin von Altrock, director of Inform's Fuzzy Technologies Division. "Motorola delivers what customers using fuzzy logic need: the performance of dedicated fuzzy hardware solutions at the price of a standard microcontroller."

Inform has added support of the 68HC11 and 68HC12 families to their fuzzyTECH product line. It provides complete fuzzy logic design software for all 68HC11 and 68HC12 derivatives. fuzzyTECH lets users design solutions based on standardized development methodology in very short time. At the push of a button, fuzzyTECH generates the fuzzy logic system as assembly code for the target microcontroller.

The unique performance/price ratio achieved by the 68HC12 executing fuzzy logic will enable new and innovative solutions in many different application areas. In hard disk design, the fuzzy logic instructions on the 68HC12 speed track-to-track seek times. In automotive applications, the 68HC12's fuzzy logic instructions will be used to make conventional anti-lock braking systems adaptive to the road surface, and to design ignition systems that deliver higher fuel efficiency. And, in telecommunications applications, fuzzy logic instructions on the 68HC12 will enable the implementation of intelligent systems for routing, cell switching, and voice compression.

On the 68HC11, fuzzyTECH emulates the entire fuzzy logic algorithm in software, while the code generated for the 68HC12 utilizes its special fuzzy logic instructions.

Together, Motorola and Inform have co-developed a low-cost version called fuzzyTECH HC11/12 Explorer. It offers the same functionality as fuzzyTECH MCU-HC11/12 Edition, but allows only developing small fuzzy logic systems.

Background Information

Benefits of Fuzzy Logic

A very reliable programming methodology, fuzzy logic also offers the benefits of simplicity, small code size, and reduced memory requirements. It allows the control of real-world systems using elements of natural language rather than rigorous mathematical modeling, thus simplifying programming. Fuzzy logic uses plain language rules and knowledge directly from the application expert rather than forcing the expert to translate knowledge into a computer programming language.

Overall, fuzzy logic needs far less memory than table-based control and, in many cases, math-based control. Fuzzy systems may use one tenth as many rules as conventional systems. Fuzzy logic reduces complexity of problems allowing simple controllers to handle complex problems or more advanced controllers to perform more quickly.

Benefits of fuzzyTECH

fuzzyTECH's support of a system using fuzzy logic does not stop with code generation. Rather, fuzzyTECH allows to monitor and modify the fuzzy logic system after implementation in real-time on the running microcontroller. Users can thus analyze the behavior of a fuzzy logic controller in real situations and optimize a running system "on-the-fly." In contrast to an in-circuit emulator, the running system is not even halted for modification of the fuzzy logic control strategy. Only a serial connection between fuzzyTECH and the microcontroller is required for this, no special hardware is needed. In addition to the serial interface, fuzzyTECH also supports the special Background Debug™ Mode interface of the 68HC12 for "on-the-fly" optimization.

Because fuzzyTECH analyzes the fuzzy logic system structure thoroughly when generating the microcontroller code, it can generate faster and more compact fuzzy logic systems than manual coding can. Even complex fuzzy logic systems compute in less than one millisecond and require less than one Kbyte of ROM space on the 68HC11. Because of the special fuzzy logic functions available on the 68HC12, implemented fuzzy logic systems will run about 15 times faster and are six times more compact than on the 68HC11.

Inform also offers a fuzzyTECH edition that supports generation of portable C code on Motorola 68HC16, 68x00, and PowerPC families. Future releases of fuzzyTECH will support the 68HC05 and 68HC08 families of microcontrollers.
