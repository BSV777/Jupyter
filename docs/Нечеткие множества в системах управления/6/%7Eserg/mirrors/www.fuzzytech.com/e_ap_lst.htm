fuzzyTECH Library: Engineering Application Papers

The books presented on the Fuzzy Logic Literature page contain a large number of case studies in various industrial and commercial application areas. These books also contain a list of references that can help you to find specific application notes. Some recent application notes are also available from this site. 2. Engineering Application Papers

1. General Application Papers

Title Abstract
Towards Fuzzy Logic Standardization In 1991, the German Association of Mechanical and Electrical Engineers (VDI/VDE) established the working group UA451 to focus on "Fuzzy Logic and Fuzzy Control" in its Measurement and Automation Chapter (GMA). This paper presents some of the intermediate results of fuzzy logic standardization activities.
Fuzzy Logic Design - Methodology, Standards, and Tools When Lotfi Zadeh introduced the world to fuzzy logic in 1965, he forgot to include, or rather was unprepared to include a user's manual. Today, this situation is much different. Fuzzy logic has become a commonly used technology, and though the large number of successful applications, development tools and methodology have evolved.

2. Engineering Application Papers

Title Abstract
Recent Successful Fuzzy Logic Applications in Industrial Automation In this paper we review eight (8) recent applications of fuzzy logic in industrial automation. All applications used the so-called "fuzzyPLC", an innovative hardware platform that merges fuzzy logic and traditional automation techniques. The fuzzy logic anti-sway crane controller discussed in this paper can be viewed on the Crane Video.
Fuzzy Logic Enhanced Control of an AC Induction Motor with a DSP In a joint application project, Texas Instruments and Inform Software used fuzzy logic to improve AC induction motor control. The results were intriguing: control performance has been improved while design effort has been significantly reduced. This application paper is also available in printed form as Application Report SPRA057 from Texas Instruments.
Optimization of a Water Treatment System This case study is about a fuzzy logic solution in biochemical production in the world's largest oral penicillin production facility in Austria. In order to reduce the energy costs of the process, the separation of water and dry substance in a decanter was optimized.
Fuzzy Logic Control of Baking TWIX Candy Bars In this paper, we present the automation of an industrial oven line employed for the continuous baking of biscuits for TWIX candy bars at MARS Corporation in Germany.
Fuzzy Logic and NeuroFuzzy Data Analysis in a Medical Shoe Application of NeuroFuzzy techniques in data analysis for a medical shoe. The entire system is integrated on a microcontroller for efficient mass-market production.
Truck Speed Limiter Control by Fuzzy Logic This case study focuses on an electro-pneumatic design of truck a speed limiter. The design of an algorithm for this control problem proved to be difficult, since the same speed limiter device is used in a variety of different trucks which exhibit different behaviors. Hence, fuzzy logic control was used to design the control algorithm.
Fuzzy Logic and NeuroFuzzy in Appliances The application of fuzzy logic and NeuroFuzzy techniques is exemplified using three (3) case studies in the area of home appliances (a/c, heating, washing).
Industrial Automation and Process Control Applications The application of fuzzy logic and NeuroFuzzy techniques is exemplified using three (3) case studies in the area of industrial automation and process control (cranes, chemical industry, sensors). The fuzzy logic anti-sway crane controller discussed in this paper can be viewed on the Crane Video.
Fuzzy Logic in Automotive Engineering The application of fuzzy logic is demonstrated using four (4) case studies in the area of automotive engineering (ABS, anti-skid control, automatic transmission, steering stabilization),.
Supervisory Control for a Coal Power Plant RWE/Rheinbraun Corporation of Germany and Inform Software Corp. have successfully used fuzzy logic supervisory control to increase efficiency and reduce air pollution in a new type of fossil fuel power plant. The plant uses a combination of a Distributed Process Control system and FactoryLink.
Practical Fuzzy Logic Design This app. note is very introductory and uses the anti-sway crane controller case study to illustrate practical fuzzy logic design and discuss the use of fuzzy logic technology versus conventional type control methods. The fuzzy logic anti-sway crane controller discussed in this paper can be viewed on the Crane Video.
Demonstration Model of fuzzyTECH Implementation on Motorola 68HC12 MCU This app. note presents a fuzzy logic demonstration model that demonstrates the use of the Motorola HC12 microcontroller, the world's first standard microcontroller with a fuzzy logic instruction set. You can also download a similar application paper from Motorola's Web Site. A PowerPoint presentation about this application is available from the Seminars and Workshops page.
Start-Up Control of a Steam Generator This app. note presents the stand-alone fuzzyTECH Runtime Module using the case study of a steam generator. You can also download the respective software simulation with the fuzzyTECH 5.0 Demo to work with the example. Also, refer to the PowerPoint Presentation about the Steam Generator.
Data Analysis of Environmental Data for Traffic Control In addition to control applications, fuzzy logic has also become a widely applied technology in data analysis and sensor signal analysis (cf. application paper Fuzzy Logic and NeuroFuzzy Data Analysis in a Medical Shoe). In this paper, the application of fuzzy data analysis is presented in traffic control systems. The traffic control system is in operation on the B27 state highway in Germany.
Antilock Braking System and Vehicle Speed Estimation Using Fuzzy Logic Siemens Corporation of Germany employed fuzzy logic in a reference design for an ABS system using the fuzzy logic coprocessor SAE81C99A (code generation for this fuzzy logic coprocessor is provided by the fuzzyTECH MCU-99 Edition). The system is discussed and the test results obtained from a BMW 328i are shown. This paper has been provided by Siemens AG.
Application of Fuzzy Control for Optimal Operation of Complex Chilling Systems This case study demonstrates the use of the Klockner-Moeller "fuzzyPLC" in a multi-variable process control application. This paper has been provided by Johnson Control in cooperation with the Polytechnical Institute of Cologne.
Monitoring Glaucoma by Means of a NeuroFuzzy Classifyer This application paper describes a combination of neural net and fuzzy logic technologies for the diagnostics of glaucoma. This paper has been provided by the German National Research Center for Environment and Health.
Fuzzy Technology Implemented in Sonar Systems This application paper describes applications of fuzzy algorithms implemented in Atlas sonar systems. From an application-oriented point of view, classification of DEMON spectra and automatic target-tracking by means of fuzzy systems are introduced and demonstrated with examples.
Development of a Fuzzy Knowledge-Based System for the Control of a Refuse Incineration Plant Refuse incineration is a complex process, whose multi-variable control problems can not be solved conventionally by deriving an exact mathematical model of the process. This application paper describes the usage of advanced fuzzy technologies within the suitable development methodology by a controller implemented for the refuse incineration plant in Hamburg-Stapelfeld, Germany.
