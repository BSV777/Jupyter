FOXBORO AND INFORM INTRODUCE FUZZY LOGIC FUNCTION
BLOCKS FOR THE FOXBORO IAS&REG AUTOMATION SYSTEM

Contact: Thomas Froese, +49-211-5966-413

DUSSELDORF (Jan. 19, 1995) -- Foxboro and Inform have announced a strategic partnership for the introduction of fuzzy logic design techniques in process control.

Foxboro, a worldwide leader in process control systems, and Inform, a worldwide leader in fuzzy logic engineering, have jointly developed fuzzy logic function blocks for Foxboro's I/A-Series automation system. The fuzzy logic function blocks base on Inform's "fuzzyTECH" line of development tools. Due to its modular structure, even existing installations can easily be enabled by this solution.

"We have selected Inform as strategic partner because of it's long-term experience in the application of fuzzy logic in real-world applications," said Hermann Dirxen, sales and marketing manager of Foxboro.

The integration of fuzzyTECH components facilitates the supervisory control of process control applications using human experience and experimental results. First joint application projects have already been implemented by Foxboro and Inform. "In a biological fermentation process of an Austrian pharmaceutics manufacturer, the integration of the fuzzyTECH component saves about $70,000 each year in energy costs," said Thomas Froese, project engineer of Foxboro.

"Because the Foxboro I/A-Series automation system is a very modern and modular architecture, a seamless integration of fuzzy function blocks is possible", said Constantin von Altrock, director of the Fuzzy Technologies division of Inform, "the fuzzy logic component centrally optimizes the setpoints and parameters of the decentral control components." In most real-world applications, fuzzy logic does not replace conventional PID-type controllers, but optimizes the operation point of the entire plant. Thus the fuzzy logic controller functions very much like a human operator that runs the process in a stable and efficient point of operation, even in changing conditions.
