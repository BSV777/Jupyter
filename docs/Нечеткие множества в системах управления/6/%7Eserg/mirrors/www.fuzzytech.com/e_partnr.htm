Texas Instruments

Some applications of fuzzy logic do require more processing power than provided by today's microcontrollers. To serve such applications, you may employ dedicated hardware solutions (such the ones we developed in cooperation with Siemens). A low-cost alternative is the use of digital signal processors (DSP) with the optimized fuzzyTECH kernel. In close cooperation with Texas Instruments, we have developed the fuzzyTECH MCU-320 Edition, which allows you to implement very fast fuzzy logic solutions on standard hardware. Check out our benchmarks for a comparison! Visit the Texas Instruments Fuzzy Logic Apps page for application examples. Applications using the fuzzyTECH MCU-320 Edition on Texas Instrument's DSP include: hard disk control, active noise cancellation, sonar signal classification, servo motor control, and many others. Note, we also have an apps paper on AC Motor Control using DSP on our web server that is also available in printed form as Application Report SPRA057 from Texas Instruments.

Siemens Semiconductor

Joining forces with Siemens Semiconductor, we created the world's first 16-bit fuzzy processor in 1992, the FUZZY166. While other fuzzy processors require a conventional host microcontroller for the non-fuzzy part of the solution, the FUZZY166 is an extended standard microcontroller with a complete instruction set. It has been designed into more than 200 products, including plastic molding machine controllers, robotics, flight control systems, and the fuzzyPLC™. By further optimizing the implementation technology, we further developed the FUZZY166 technology to a "software-only" solution for the entire range of Siemens 16-bit microcontrollers. The fuzzyTECH MCU-166 Edition supports both FUZZY166 and standard 80C165/6/7 devices. Because of the high efficiency of the FUZZY166 technology, the implementation reaches a very high computational performance. Check out our benchmarks for a comparison! For low-cost / high volume applications, we support the 8 bit fuzzy coprocessor unit 81C99A, today's fastest hardware platform for large fuzzy logic systems. The fuzzyTECH 81C99A Hardware Package comes with complete software and hardware debugging tools. For a listing of the fuzzy software tools available for Siemens microcontrollers, refer to the Siemens web server's MCU Tool Listing.

Intel

About 10 years ago, a straightforward implementation of the fuzzy logic algorithm required about 1 second computing time for a small fuzzy logic system on a standard 8051 microcontroller. Such low level of performance rendered the use of fuzzy logic impossible in most real-time applications. Two solutions existed to this problem: one was to accelerate the fuzzy logic computation in hardware (i.e., creating fuzzy processors); and the other one was to accelerate the fuzzy logic computation in software. With our partner Intel, we decided to go with the second alternative when we started a research project in 1992 to accelerate the fuzzy logic algorithm in software. The progress was dramatic: in 1994, we were able to compute the same fuzzy logic system on a 8051 in just 10 milliseconds, and today's fuzzyTECH MCU-51 Edition implements it with only 1 millisecond computing time. This is equal to an acceleration of about 1000x (!) and sufficient for most real-time control applications. For more details, check the Electronic Engineering Times article we provide on this web server. Today, we support all Intel microcontroller families with the fuzzyTECH MCU-96 and MCU-51 Editions. Applications using the fuzzyTECH MCU-96 and MCU-51 Edition on Intel's MCUs include: hard disk control, anti-skid braking systems, CNC positioning control, metal melting ovens, home appliances control, voice command systems, handwritten language recognition, and many others. Visit Intel's Web Site and view an app note about how fuzzyTECH was used in an anti-lock braking system (ABS), or download a general Intel paper about the Fuzzy Logic Control with the Intel 8XC196 Embedded Microcontroller. More applications of fuzzy logic using Intel's MCUs have been published by Intel in a book "Fuzzy Logic Applications Handbook" (Intel Order Number 272589-001).

Klockner-Moeller

By teaming up with Klockner-Moeller Corporation, a leading manufacturer of compact programmable logic controllers (PLC), we have created the so-called "fuzzyPLC™". The fuzzyPLC combines conventional discrete and analog control techniques (IEC1131) with fuzzy logic. Using the FUZZY166 technology developed by Siemens Semiconductor and Inform plus an analog ASIC, a two-chip solution was created that provides an unsurpassed performance/cost ratio for industrial automation solutions. Check out our benchmarks for a comparison! The fuzzyPLC model PS4/401MM2 hardware features 12bit A/D&D/A ,as well as digital interfaces, integrated field bus support, local expandability with i/o modules, memory card management, and 512 KB main memory. The fuzzyPLC software is the SUCOsoft fuzzyTECH S40 Edition, a specialized version of fuzzyTECH that combines IEC1131 programming with the standard functionality of fuzzyTECH in a single piece of software. Both hardware and software are currently available only trough Klockner-Moeller. Check out the Klockner-Moeller Home Page for the contact nearest to you. Note, we have on our web server an application paper featuring 8 case studies of fuzzyPLC applications.
