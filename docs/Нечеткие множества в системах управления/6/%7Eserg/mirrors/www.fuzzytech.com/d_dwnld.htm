Fuzzy Logic Benchmarks Herunterladen

Zum Vergleich der Rechenleistung auf verschiedenen Hardwareplattformen werden in allen Rechenbereichen Leistungsmaße, sogenannte Benchmarks, verwendet. Für Fuzzy Logic Systeme wurden u.a. solche Maße wie MFLIPS (million fuzzy logic inferences per second) eingeführt. Es hat sich jedoch gezeigt, daß diese Leistungsmaße nur einen begrenzten praktischen Wert haben. Eine MFLIPS-Zahl gibt z.B. keine befriedigende Auskunft darüber, wieviel Rechenzeit ein spezielles Fuzzy Logic System für eine bestimmte Hardwareplattform benötigt. Da viele verschiedene Definitionen darüber existieren, was "eine Fuzzy Logic Inferenz" ist und welche Rechenschritte sie beinhaltet, gingen einige Verkäufer soweit, nur eine einzige Multiplikation oder minimale Berechnung als "eine Fuzzy Logic Inferenz" zu zählen. Weiterhin kann bei einigen Fuzzy Logic Systemen die Fuzzifikation und Defuzzifikation weitaus mehr Rechenzeit in Anspruch nehmen als die Regelinferenz.

Um ein praktischeres Maß zu finden, wurde von einem Industriekonsortium eine Reihe "typischer Fuzzy Logic Systeme" definiert, die das gesamte Spektrum von einfachen bis komplexen Systemen sowie verschiedene Anwendungsbereiche abdeckt und gegenwärtig als Standard-Benchmark-Reihe für Fuzzy Logic Systeme vom IEEE Standing Committee on Standards vorgeschlagen wird. Mitglieder des Industriekonsortiums sind solche Firmen wie Inform, Intel, Klockner-Moeller, Microchip, Motorola, Rockwell, Siemens HL (alphabetische Reihenfolge).

Die vorgeschlagene Benchmark-Reihe ermöglicht den Vergleich der Rechenzeit und benötigtem Codespeicherplatz für "typische Fuzzy Logic Systeme" verschiedener Größe und Anwendungsgebiete. Anwender, die eine ballpark-Schätzung der Rechenressourcen zur Verarbeitung ihrer Fuzzy Logic Lösung benötigen, können der Benchmark-Reihe das am besten zu ihrer Anwendung passende Fuzzy Logic System entnehmen. Die gesamte Rechenzeit und der benötigte Speicherplatzbedarf der "typischen Fuzzy Logic Systeme" wird somit zu einem sinnvollen Maß zum Leistungsvergleich verschiedener Hardwareplattformen. Damit gewährleistet werden kann, daß die Benchmarkergebnisse faire Vergleiche darstellen, ist ein Definieren der Testprocedur erforderlich (Auflösung, Eingabemuster, etc...).

Weiterer Information über die FuzMarks Benchmarks sind in dem Buch "Fuzzy Logic -Band 1: Technologie" enthalten.

Dieses Buch enthält auch eine komplette Referenz und Beispiele für das FTL-Format. Die vollständige Benchmarkreihe wird als FTL-Datei ebenfalls mit jeder fuzzyTECH Edition ausgeliefert und ist auf der Installations-CD-ROM im Unterverzeichnis ..\SAMPLES\BENCHxx zu finden.

Die Nutzung der 5 Fuzzy Logic Standard Benchmarks (FuzMarks) ist unter folgenden Bedingungen frei:

  • Wenn Sie eigene Ergebnisse publizieren, die auf FuzMarks basieren, müssen Sie folgende Informationen angeben: Genaue Spezifikation der Zielplattform (inklusive Taktfrequenz, Art des Speicherzugriffs,..), maximale und durchschnittliche Rechenzeit für das gesamte Bechmarksystem gestestet mit einem vollständigen 10% Eingangsmuster, Speicherbedarf (RAM und ROM) sowie verwendete Registerspeicher und benutzten Compiler.
  • Angabe der verwendeten FuzMarks sowie deren Copyright sowie der Bezugsquelle (dieser WWW-Server). Außerdem schicken Sie uns bitte eine Kopie Ihrer Publikation zu.
  • Sie dürfen die FuzMarks nicht weitergeben, allerdings können Sie diesen WWW-Server für jeden als Bezugsquelle nennen.
