ЧИСЛЕННАЯ ОЦЕНКА ОДНОГО МЕТОДА ФИЛЬТРАЦИИ ИЗМЕРЕНИЙ


Васильев А.Б., Кряжевских С.В.
ОАО "Отделение разработки систем"

 

   Одной из существенных задач для получения корректных результатов измерений является фильтрация помех, наводок, динамических ударов и других случайных составляющих, возникающих в измерительном тракте автоматизированной системы управления. Поэтому во многих инструментальных системах проектирования АСУ ТП есть встроенные средства фильтрации аналогового входного сигнала. Например, в отечественной SCADA-системе ТРЕЙС МОУД реализованы следующие методы:

  • введение апертуры (зоны нечувствительности);
  • введение инерционного звена (экспоненциального сглаживания);
  • метод скользящего среднего;
  • усреднение за заданный период.
   С другой стороны, при проектировании измерительных каналов АСУ ТП приходится одновременно решать две основные задачи:
  • фильтрации "зашумленного" сигнала, мешающего текущим наблюдениям;
  • получения достоверной динамики постоянной составляющей сигнала, необходимой для "ручной" или автоматической выработки управляющего воздействия.


   Отсюда, для достаточно точного отслеживания медленно меняющегося входного сигнала наибольший интерес представляет метод скользящего среднего или метод фильтрации с конечной памятью. Эффективность данного метода была апробирована на выборке измерений давления в тепловой сети города Кирова. Численные значения снимались с интервалом 1 минута после предварительного их усреднения за этот же период.

   Исходные значения выборки показывали значительную зашумленность, практически исключающую сделать какие-либо определенные выводы о
поведении параметра за рассматриваемый отрезок времени. После сглаживания скользящим средним "по 10 последним значениям" зашумленность в выборке заметно убавилась. При этом явно проявились участки подъема и спада величины сигнала - изменения постоянной составляющей. Для оценки случайной составляющей сигнала из сглаженной выборки было удалено два участка с ненулевым отклонением от среднего - с постоянной составляющей.

   Результаты статистического анализа использованы для реализации алгоритмов статистической фильтрации измерений в локальном контуре системы автоматического управления центральными тепловыми пунктами города Кирова.
