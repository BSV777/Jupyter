  В теории распознавания образов обычно рассматривается задача кластеризации с одним критерием выделения кластеров. В данной работе предложено обобщение задачи, когда кластеры формируются по нескольким разнородным критериям.
  В качестве метода многокритериальной оценки используется подход Беллмана-Заде, в котором каждому критерию сопоставляется нечеткая функция принадлежности, а в качестве оценки принадлежности точки к кластеру принимается минимум из значений функций принадлежности по каждому из критериев. Оценка принадлежности к кластеру, используемая в стандартном методе кластеризации ISODATA, заменена нечеткой оценкой. Такой подход позволяет рассмотреть широкий спектр задач, связанных с выделением кластеров по набору параметров. Предложенная методика опробована на тестовой задаче кластеризации множества цветных точек по критериям расстояния, цвета и размера точек.
