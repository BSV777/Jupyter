Для решения различных прикладных задач с использованием аппарата нечеткой логики
 предлагаются следующие услуги:
- единичные консультации или семинары с изложением теоретических основ современных
технологий управления с демонстрацией возможностей этих технологий на примерах решения
экономических или производственных задач;
- выполнение отдельных работ по контрактам или трудовым соглашениям;
- выполнение долгосрочных программ.
 

Предлагаются следующие материалы по теории и прикладным аспектам использования фази логики:

1. Отчет " Методы, средства и практическая реализация фази управления"

2. Перевод книги Rene Jager "Fuzzy  Logic in Control"

3. CD-диск с набором программ по нечеткой логике.
